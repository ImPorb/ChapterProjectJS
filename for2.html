JavaScript Infinite for løkke


Hvis testtilstanden i en for løkke er alltid true den går for alltid (til minnet er fullt). For eksempel, 


            // infinite for loop
            for(let i = 1; i > 0; i++) {
            // block of code
            }

I programmet ovenfor er betingelsen alltid true som da vil kjøre koden i uendelig tid.
